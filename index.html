About DepShield

DepShield is an open-source supply chain dependency security scanner that detects three critical attack vectors that threaten modern software projects.

🔤

Typosquatting

Detects packages with names suspiciously similar to popular packages — a common vector for injecting malicious code via misspelled dependency names.

lodahs → lodash
🔀

Dependency Confusion

Identifies packages at risk of namespace confusion attacks, where a public package could override an intended private/internal package.

internal-auth → public registry risk
⚠️

Abandoned Packages

Flags dependencies showing signs of abandonment — unmaintained packages are prime targets for supply chain hijacking.

0.0.1 → single-version, no updates
